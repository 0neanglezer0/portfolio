# 프로필 이미지 가이드

이 폴더에 프로필 사진을 추가해주세요.

## 필요한 파일

- **파일명**: `profile.jpg`
- **용도**: About 페이지의 프로필 사진

## 이미지 사양

- **형식**: JPG 또는 PNG
- **권장 크기**: 500x500px 이상 (정사각형)
- **비율**: 1:1 (정사각형)
- **파일 크기**: 500KB 이하 권장
- **품질**: 고화질, 선명한 얼굴 사진

## 프로필 사진 팁

1. **조명**: 자연광이나 밝은 조명에서 촬영
2. **배경**: 단색 배경 또는 깔끔한 배경 추천
3. **표정**: 자연스럽고 친근한 표정
4. **복장**: 프로페셔널한 복장 권장
5. **구도**: 상반신 또는 얼굴 중심 구도

## 없을 경우

프로필 이미지가 없어도 웹사이트는 정상 작동합니다.
회색 배경(placeholder)이 표시됩니다.

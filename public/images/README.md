# 이미지 파일 가이드

포트폴리오에 필요한 이미지들을 아래 폴더에 넣어주세요.

## 📁 폴더 구조

```
/public/images/
├── profile/
│   └── profile.jpg          # 프로필 사진 (정사각형 추천, 500x500px 이상)
├── hero/
│   └── hero-bg.jpg          # 히어로 섹션 배경 (옵션, 1920x1080px 추천)
└── projects/
    ├── skt-ai-frontier.jpg  # SKT AI Frontier 프로젝트 이미지
    ├── skku-admissions.jpg  # 입학전략설명회 프로젝트 이미지
    ├── marketing-automation.jpg  # 마케팅 캠페인 자동화 이미지
    └── overseas-camp.jpg    # 해외 청소년 교육 프로그램 이미지
```

## 📐 권장 사이즈

- **프로필 사진**: 500x500px (정사각형)
- **프로젝트 썸네일**: 1200x800px (3:2 비율)
- **히어로 배경**: 1920x1080px (옵션)

## 📝 파일 형식

- JPG 또는 PNG 형식
- 웹 최적화된 파일 (각 파일 500KB 이하 권장)

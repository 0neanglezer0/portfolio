## 이미지 사양

- **형식**: JPG 또는 PNG
- **권장 크기**: 1200x800px (3:2 비율)
- **파일 크기**: 각 파일 500KB 이하 권장 (웹 최적화)
- **품질**: 고화질, 선명한 이미지

## 이미지 준비 팁

1. **저작권**: 본인이 촬영하거나 사용 권한이 있는 이미지만 사용
2. **Unsplash/Pexels**: 무료 고품질 이미지 사이트 활용 가능
3. **AI 이미지 생성**: Midjourney, DALL-E, Stable Diffusion 등 활용
4. **스크린샷**: 프로젝트 결과물, 도구 화면 등 캡처
5. **이미지 편집**: Figma, Canva 등으로 썸네일 디자인

## 없을 경우

이미지가 없어도 웹사이트는 정상 작동합니다.
기본 그라데이션 배경이 표시됩니다.

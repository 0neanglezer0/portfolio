# 서울의 보이지 않는 벽 - 생활 서비스 접근성 불평등

## 📌 프로젝트 개요

**목표**: 서울시 지역별 생활 서비스 접근성 격차를 AI로 분석하고, 최적 입지 추천 솔루션을 제시하는 3분 영상 콘텐츠 제작

**핵심 메시지**: 같은 서울이지만, 어디 사느냐에 따라 병원/은행/관공서까지 10분 vs 50분의 시간 격차 존재

---

## 🎯 영상 구성 (3분)

### ACT 1: 문제 제기 (0:00-0:45)
- 오프닝: 서울 지도가 점점 확대되며 "24km의 서울, 하지만..."
- 두 인물의 하루 비교
  - 강남 거주 A씨: 도보 10분 내 모든 서비스 가능
  - 외곽 거주 B씨: 필수 서비스 접근에 평균 40분 이상
- 데이터 시각화: 지역별 접근성 히트맵 등장
- 나레이션: "서울에는 보이지 않는 벽이 있습니다"

### ACT 2: 데이터 분석 (0:45-1:45)
- AI 대시보드 등장
- 핵심 지표 시각화
  1. 의료시설 접근성 맵
  2. 금융시설 접근성 맵
  3. 행정시설 접근성 맵
  4. 종합 접근성 점수
- 취약 지역 하이라이트
- 고령층/장애인에게 더 심각한 불평등 지적

### ACT 3: AI 솔루션 (1:45-2:45)
- AI 기반 해결 방안 3가지
  1. 실시간 접근성 히트맵
  2. 최적 서비스 입지 추천 알고리즘
  3. 개인 맞춤 생활권 분석
- 솔루션 적용 시뮬레이션
- Before/After 비교

### ACT 4: 클로징 (2:45-3:00)
- 메시지: "AI가 찾는 더 공평한 서울"
- CTA: 대시보드 웹사이트 소개
- 엔딩 크레딧

## 📊 데이터 수집 계획

### 필요 데이터셋

#### 1. 서울시 행정구역 데이터
- 출처: 서울 열린데이터광장
- 내용: 25개 자치구, 424개 행정동 경계 데이터 (GeoJSON)

#### 2. 의료시설 데이터
- 병원/의원 위치 (위도, 경도)
- 시설 규모 (종합병원, 병원, 의원)
- 진료과목

#### 3. 금융시설 데이터
- 은행/ATM 위치
- 운영시간

#### 4. 행정시설 데이터
- 주민센터, 구청 위치
- 우체국, 경찰서, 소방서

#### 5. 대중교통 데이터
- 지하철역 위치
- 버스정류장 위치

#### 6. 인구 데이터
- 행정동별 인구 수
- 연령대별 분포
- 1인 가구 비율

1. **서울 열린데이터광장** (data.seoul.go.kr)

   - 공공시설 정보

   - 시설 위치 정보
   - 거리/시간 계산

4. **국토교통부 공간정보**
   - 도로망 데이터

## 🤖 AI 모델 및 알고리즘

### 1. 접근성 점수 계산 알고리즘
```python
접근성_점수 = (
    의료시설_접근성 * 0.4 +
    금융시설_접근성 * 0.2 +
    행정시설_접근성 * 0.2 +
    대중교통_접근성 * 0.2
)
```

### 2. 최적 입지 추천 AI
- 머신러닝 모델: K-means 클러스터링
- 목표: 접근성 격차 최소화하는 신규 시설 위치 추천
- 제약조건: 예산, 토지 가용성

### 3. 개인 맞춤 분석
- 사용자 입력: 현재 위치, 필요 서비스, 이동 수단
- 출력: 30분 생활권 내 이용 가능 서비스 맵

## 🔑 필요한 API 키

### 1. 카카오 로컬 API (필수)
- **용도**: POI 검색, 거리/시간 계산
- **등록**: https://developers.kakao.com/
- **무료 할당량**: 일 300,000건
- **기능**:
  - 키워드 장소 검색
  - 좌표로 주소 변환
  - 거리/시간 계산 (Matrix API)

### 2. 공공데이터포털 API (필수)
- **용도**: 의료기관, 금융기관, 우체국 정보
- **등록**: https://www.data.go.kr/
- **무료**: 서비스별 트래픽 제한 있음
- **필요 API**:
  - 의료기관 정보 조회
  - 금융기관 영업점 정보 조회

### 3. 서울 열린데이터광장 API (선택)
- **용도**: 실시간 서울시 데이터
- **등록**: https://data.seoul.go.kr/
- **무료**: 대부분 인증키 불필요
- **주로 다운로드 방식 사용**

- 추가적인 시설 정보

## 🛠️ 데이터 수집 도구

### Python 라이브러리
```python
# 설치 필요
requests          # API 호출
pandas           # 데이터 처리

geopy            # 좌표 계산
```

## 📝 데이터 수집 스크립트 구조

### 1. `collect_static_data.py`
- 서울시 오픈 데이터 다운로드
- GeoJSON, CSV 파일 저장

### 2. `collect_kakao_poi.py`
- 카카오 로컬 API로 POI 수집
- 병원, 은행, 편의점 등

### 3. `collect_public_api.py`
- 공공데이터포털 API 호출
- 의료기관, 금융기관 상세 정보

### 4. `merge_data.py`
- 수집한 데이터 병합
- 중복 제거, 좌표 보정

### 5. `calculate_accessibility.py`
- 행정동별 접근성 점수 계산
- 최근접 시설까지 거리/시간 계산

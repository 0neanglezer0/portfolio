# 서울의 보이지 않는 벽 - 영상 스크립트

**제목**: 서울의 보이지 않는 벽: AI가 발견한 생활 접근성 불평등
**길이**: 3분 (180초)
**형식**: 데이터 기반 다큐멘터리 + AI 솔루션 제안
**타겟**: 서울시 AI 영상 공모전

---

## 🎬 ACT 1: 문제 제기 (0:00 - 0:45, 45초)

### SCENE 1: 오프닝 (0:00 - 0:15)

**비주얼**:
- 서울 위성 사진이 서서히 확대
- 반짝이는 불빛들이 점점 선명해짐
- 화면이 25개 자치구로 나뉘며 색깔별로 분리

**나레이션**:
```
서울.
동서 24km, 남북 30km.
같은 도시, 같은 하늘 아래.

하지만 당신이 사는 곳에 따라,
전혀 다른 서울이 펼쳐집니다.
```

**음악**: 미니멀한 피아노 (긴장감)

---

### SCENE 2: 두 사람의 하루 (0:15 - 0:35)

**비주얼**:
- 화면 분할 (좌: 송파구, 우: 금천구)
- 애니메이션으로 두 사람의 동선 표시
- 시계 타이머가 각각 카운트

**좌측 화면 - 송파구 A씨**:
- 아파트 → 병원 (도보 5분)
- 아파트 → 은행 (도보 3분)
- 아파트 → 구청 (도보 7분)
- **총 소요 시간: 15분**

**우측 화면 - 금천구 B씨**:
- 집 → 병원 (버스 25분)
- 집 → 은행 (도보 15분)
- 집 → 구청 (버스 20분)
- **총 소요 시간: 60분**

**나레이션**:
```
송파구에 사는 A씨.
병원, 은행, 구청까지 평균 도보 5분.

금천구에 사는 B씨.
같은 일을 하는데 평균 20분 이상.

하루 1시간, 일주일이면 7시간.
한 달이면 하루를 통째로 길 위에서 보냅니다.
```

**데이터 표시**:
- 송파구 접근성 점수: 67.1점
- 금천구 접근성 점수: 13.7점
- **격차: 4.9배**

---

### SCENE 3: 문제 정의 (0:35 - 0:45)

**비주얼**:
- 서울 지도가 히트맵으로 변환
- 빨간색(낮음) ~ 파란색(높음) 그라데이션
- 강남권은 파란색, 외곽은 빨간색

**나레이션**:
```
이것이 서울의 '보이지 않는 벽'입니다.
생활 서비스 접근성 격차.

AI가 분석한 결과,
25개 자치구 중 44%가 F등급.
평균 점수는 겨우 38.8점.
```

**화면 하단 자막**:
- 분석 대상: 병원 409개, 은행 324개, 행정시설 402개, 지하철역 128개
- AI 분석: 1,263개 시설 × 25개 자치구

## 🎬 ACT 2: 데이터 분석 (0:45 - 1:30, 45초)

### SCENE 4: 의료 접근성 (0:45 - 0:58)

**비주얼**:
- 서울 지도 위에 병원 아이콘이 플롯됨
- 밀집도가 색깔로 표현
- 강남 3구 vs 외곽 비교

**나레이션**:
```
첫 번째, 의료 접근성.

강남, 서초, 송파.
10분 안에 병원 5개 이상.

하지만 금천, 강북, 도봉.
가장 가까운 병원까지 평균 20분 이상.
```

**차트 표시**:
```
TOP 3: 송파 (82.1점), 양천 (86.2점), 중랑 (82.2점)
BOTTOM 3: 금천 (4.8점), 용산 (7.5점), 강북 (11.1점)
```

---

### SCENE 5: 금융 접근성 (0:58 - 1:11)

**비주얼**:
- 은행/ATM 위치가 지도에 표시
- 특히 강북 외곽 지역의 공백이 두드러짐

**나레이션**:
```
두 번째, 금융 접근성.

은행이 없어 ATM을 찾아 헤매는 곳.
현금 인출 하나에 30분을 쓰는 어르신들.

서울 전체 은행의 60%가
강남 5개 구에 집중되어 있습니다.
```

**데이터 표시**:
- 강남구: 32개 은행/ATM (금융 점수 77.0)
- 강북구: 4개 은행/ATM (금융 점수 0.8)

---

### SCENE 6: 교통 접근성 (1:11 - 1:24)

**비주얼**:
- 지하철 노선도 오버레이
- 역세권(500m) 원이 표시됨
- 사각지대가 빨간색으로 하이라이트

**나레이션**:
```
세 번째, 대중교통 접근성.

지하철 역세권 500m 안.
서울 시민의 절반은 살고,
나머지 절반은 살지 못합니다.

특히 강서, 은평, 노원 외곽.
지하철까지 버스로 30분.
```

**차트 애니메이션**:
- 역세권 거주 비율: 강남 78% vs 외곽 평균 42%

---

### SCENE 7: 종합 분석 (1:24 - 1:30)

**비주얼**:
- 4개 점수(의료, 금융, 교통, 행정)가 레이더 차트로
- 송파구 vs 금천구 비교
- 금천구의 차트는 거의 중심에 가까움

**나레이션**:
```
결과적으로,
어디에 사느냐가 당신의 시간을,
당신의 기회를,
당신의 삶의 질을 결정합니다.
```

**화면 전환**: 히트맵이 점점 어두워지며 암전

---

## 🎬 ACT 3: AI 솔루션 (1:30 - 2:30, 60초)

### SCENE 8: AI 분석 시스템 (1:30 - 1:45)

**비주얼**:
- 어두웠던 화면이 밝아지며 AI 대시보드 등장
- 실시간 데이터가 흐르는 인터페이스
- "AI Accessibility Analyzer" 타이틀

**나레이션**:
```
하지만 문제를 발견했다면,
해결책도 있습니다.

AI가 제시하는 세 가지 솔루션.
```

**화면**: 3개의 아이콘이 나타남
1. 🗺️ 실시간 접근성 지도
2. 📍 최적 입지 추천
3. 👤 개인 맞춤 분석

---

### SCENE 9: 솔루션 1 - 실시간 접근성 지도 (1:45 - 1:58)

**비주얼**:
- 인터랙티브 지도가 작동하는 모습
- 사용자가 클릭하면 주변 시설이 표시
- 거리, 시간, 경로가 실시간 계산

**나레이션**:
```
첫 번째, 실시간 접근성 지도.

당신의 위치를 입력하면,
30분 안에 갈 수 있는 모든 시설을 보여줍니다.

병원, 은행, 관공서, 지하철역.
한눈에 파악하고, 최적 경로를 찾습니다.
```

**화면 시연**:
- 금천구 한 지점을 클릭
- 30분 생활권이 원으로 표시
- 접근 가능한 시설이 핀으로 표시

---

### SCENE 10: 솔루션 2 - 최적 입지 추천 (1:58 - 2:13)

**비주얼**:
- 서울 지도 위에 AI가 계산한 후보지가 표시
- 각 후보지마다 "접근성 개선률" 수치가 표시
- 시뮬레이션: 신규 시설 배치 전/후 비교

**나레이션**:
```
두 번째, AI 최적 입지 추천.

새로운 병원이나 공공시설을 어디에 지어야
가장 많은 시민에게 혜택이 갈까?

AI가 분석한 결과,
금천구 시흥동에 종합병원 1개를 지으면
주변 10만 명의 의료 접근성이 40% 개선됩니다.
```

**시뮬레이션 화면**:
- BEFORE: 금천구 의료 점수 4.8점
- 신규 병원 추가
- AFTER: 금천구 의료 점수 23.5점 (+389%)

---

### SCENE 11: 솔루션 3 - 개인 맞춤 분석 (2:13 - 2:30)

**비주얼**:
- 사용자 프로필 입력 화면
- 나이, 거주지, 이동 수단 선택
- AI가 맞춤형 리포트 생성

**나레이션**:
```
세 번째, 개인 맞춤 생활권 분석.

고령자라면 가까운 병원이 중요하고,
직장인이라면 은행과 지하철이 중요합니다.

AI는 당신의 상황에 맞춘
최적의 거주지를 추천하고,
현재 위치의 개선 방안을 제시합니다.
```

**화면 예시**:
```
[65세, 금천구 거주, 도보 이동]
→ AI 추천: 가산동 → 독산동 이전 시
   의료 접근성 +35%
   또는
   금천구청역 인근 신규 보건소 건립 건의
```

서울의 보이지 않는 벽.

AI가 데이터를 분석하고,

AI가 더 공평한 서울을 설계합니다.

어디에 살든,

## 💡 제작 시 주의사항

1. **데이터 신뢰성 강조**
   - 실제 분석 기반임을 명확히 표시
   - 샘플 데이터 사용 시 "샘플 데이터 기반 시뮬레이션" 표기

2. **감성 + 데이터 밸런스**
   - 너무 차갑지 않게 (사람들의 삶)
   - 너무 감성적이지 않게 (객관적 데이터)

3. **AI 기술 강조**
   - 단순 데이터 시각화가 아닌 AI 분석임을 부각
   - 머신러닝, 예측 모델 등 AI 용어 자연스럽게 삽입

4. **실용성 어필**
   - 실제 사용 가능한 대시보드
   - 정책 제안 자료로 활용 가능
   - 시민들이 직접 확인 가능

5. **시각적 임팩트**
   - 3분 안에 메시지 전달
   - 화려하지만 깔끔한 디자인
   - 색상 일관성 유지 (빨강=나쁨, 파랑=좋음)
